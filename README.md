# Gitbook Explorer
This app supposed to be a Gitbook Explorer.  
The features are as below:
1. Explore(finished),sort(unfinished)
2. Read(processing)
3. Download PDF(processing)
4. Search(unfinished)
5. Topics(unfinished)
6. Login(Further, and star, subscribe also)
7. Forget, Settings or localstorage?
